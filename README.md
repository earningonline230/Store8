# Cosmetics e-Shop Application

### HTML & CSS Course

> Cosmetic e-Store  made with HMTL and CSS (Flexbox)

The project is build for the HTML & CSS Capstone Project is based on an cosmetic store.

## Built With

 - HTML
 - CSS
 - Flexbox
 - Font-awesome
 - Git-Flow
- Reponsive Design
